Detailed Description

Concept for reading and writing characters.

For read-only stream, no need to implement PutBegin(), Put() and PutEnd().

For write-only stream, only need to implement Put().

concept Stream {
typename Ch;
Ch Peek() const;
Ch Take();
size_t Tell();
Ch* PutBegin();
void Put(Ch c);
size_t PutEnd(Ch* begin);
}
