Welcome to our graph database

Database FAQ

  • The database is implemented in C and C++
  • The database uses the Mongoose open source web server in C
  • All queries are written in C++, directly into the application. This makes the queries themselves extremely fast, compared to quering another database
  • The database defaults to port 8000
  • It uses d3.js to render the graph
  • Custom DatabaseLoader class should be written to handle data, that is the only class that needs to be modified
  • The program requires the necessary properties of nodes and edges to be set to render the graph:
Nodes:
name: <node name>,
group : <integer>
Edges
source: <Node ID>,
target: <Node ID>,
value: <integer>

API instructions

  • See this screen: 
    http://127.0.0.1:8000
  • Display the forced directed graph: 
    /graph e.g http://127.0.0.1:8000/graph
  • Display graph json
    /showAll e.g http://127.0.0.1:8000/showAll
    If the length parameter is passed then then number of nodes returned can be limited
    /showAll?length=10 e.g http://127.0.0.1:8000/showAll?length=10
  • Return all nodes: 
    /showAllNodes e.g http://127.0.0.1:8000/showAllNodes
  • Return all edges: 
    /showAllEdges e.g http://127.0.0.1:8000/showAllEdges
  • Return a node with an ID: 
    /showNode?id=[ID] e.g http://127.0.0.1:8000/showNode?id=web
  • Return an edge with an ID: 
    /showEdge?id=[ID] e.g http://127.0.0.1:8000/showEdge?id=12345
